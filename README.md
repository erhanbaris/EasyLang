# EasyLang
It is a programming language that has begun to be developed for testing purposes.

Assignment operations
```
data1 = 1
data2 = true
data3 = .555
data4 = "hello world"
data5 = 1024
```

Function call
```
print data1
print (data1, data2, data3, data4, data5)
```

If statement
```
if data1 > 1 then
    print "bigger"
else
    print "opssss"

if data1 == 1 then
    print "equal"
else
    print "not equal"
```

Loop
```
i -> 0..100

}
```
